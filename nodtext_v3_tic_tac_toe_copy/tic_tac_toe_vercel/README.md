# Tic Tac Toe (Vercel Ready)

A standalone, browser-based Tic Tac Toe game designed for gaze-controlled interaction, optimized for deployment on Vercel.

## Features
- **Gaze Tracking Ready:** Integrated with MediaPipe for eye-tracking control.
- **Responsive Design:** Optimized for full-screen browser usage.
- **Static Deployment:** Ready for Vercel, Netlify, or GitHub Pages.

## How to Run Locally
1. Navigate to the directory.
2. Run a local server: `python -m http.server 8000`.
3. Open `http://localhost:8000` in your browser.
