# Tic-Tac-Toe Game

This is a simple, text-based Tic-Tac-Toe game implemented in Python.

## How to Play

1.  Run the game from your terminal:
    ```sh
    python tic_tac_toe.py
    ```
2.  Players take turns entering the row and column (0-2) where they want to place their mark.
3.  The first player to get three of their marks in a row, column, or diagonal wins.
4.  If all spaces are filled and no one has won, the game is a draw.
